Como criar esse "Botão Mágico" na IDE

Respondendo à sua pergunta: Não existe uma skill pronta do tipo "instalar e clicar" que adivinhe os benefícios de negócio do seu código de forma automática. No entanto, com a arquitetura MCP em IDEs modernas (como Cursor ou Windsurf), você pode montar esse pipeline.

LLM (Claude/GPT)

Lê a árvore DOM e mapeia a função técnica para um Benefício de Negócio

Passo a Passo da Solução:

  1. Construa um Servidor MCP de Automação: Você precisará de um script MCP (em Python ou Node.js) que integre o Playwright. O MCP exporá uma "ferramenta" (tool) para a sua IDE chamada scan_frontend_components.
  2. Prompt na IDE: No Cursor, você daria o comando: "Use a ferramenta scan_frontend_components para varrer o Ethos Flow no localhost:3000. Para cada botão e tela encontrada, gere uma tabela em Markdown traduzindo a ação técnica em um benefício comercial para o dono da clínica."
  3. A Mágica da IA: A IA não sabe o que o botão faz apenas olhando o HTML bruto, mas ela infere: Se o Playwright achou um botão chamado "Gerar Recibo de Sessão", o LLM traduz para "Benefício: Conformidade fiscal e economia de 5min por paciente para o psicólogo."
  4. Resultado: A IDE gera um relatório completo automaticamente para você.
